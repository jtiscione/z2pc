# z2pc
Yet another Mandelbrot Set zoomer.

###Version
1.0.0

###Installation

z2pc developed on node.js v. 6.9.1.

```sh
$ npm install
```
installs dependencies (Webpack, Babel, RxJS 5, etc.)

### Compile
To compile all HTML, CSS, JS to dist directory
```sh
$ webpack
```

or
```sh 
$ webpack -w
```

For production
```sh
$ webpack --release
```
